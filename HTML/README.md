# HTML相关

HTML，全称 HyperText Markup Language，译称：超文本标记语言。

#### 基本骨架结构
```html
<!-- 文档类型声明 -->
<!DOCTYPE html>
<!-- 根元素 -->
<html>
  <!-- 头部元素容器 -->
  <head>
    <!-- meta为元数据，其中charset定义文档的字符编码 -->
    <meta charset="utf-8">
    <!-- 文档标题，必需的 -->
    <title>untitle</title>
  </head>
  <!-- 文档主题内容 -->
  <body>
    hello world
  </body>
</html>
```
